
# 📱 Mini Catálogo de Celulares

## 📖 Sobre o Projeto
O **Mini Catálogo de Celulares** é um projeto web simples e direto, desenvolvido para apresentar uma lista de smartphones disponíveis para visualização. A aplicação lista diferentes modelos de celulares, exibindo o nome, preço e uma breve descrição das principais características de cada aparelho.

Este projeto tem como objetivo principal praticar a estruturação de páginas web e servirá como base para futuras implementações de estilização e interatividade.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do catálogo e dos produtos.
* **CSS3:** Estilização visual e layout (via arquivo `estilo.css`).
* **JavaScript:** Lógica de interatividade (via arquivo `script.js`).
